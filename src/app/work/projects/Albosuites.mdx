## Descripción General

Participé en la creación de contenido digital para Almacopio, un proyecto inmobiliario ubicado en La Alborada, Guayaquil. Bajo la estrategia "Tu hogar más cerca", el objetivo fue destacar la ubicación estratégica del proyecto, resaltando su cercanía con restaurantes, hospitales, escuelas, universidades y centros comerciales. A través de contenido visual atractivo y dinámico, se buscó transmitir los beneficios de vivir en una zona con todo al alcance.

![Albosuites content 2](/images/projects/project-01/Albosuites02.jpg)

## Retos Principales

- **Comunicar la ventaja de la ubicación**: Mostrar visualmente cómo el proyecto facilita el acceso a múltiples servicios esenciales.

- **Crear contenido atractivo y claro**: Desarrollar materiales digitales que fueran informativos y visualmente atractivos para potenciales compradores.

- **Adaptabilidad a diferentes plataformas digitales**: Asegurar que el contenido fuera efectivo tanto en redes sociales como en otros medios digitales.

![Albosuites content 3](/images/projects/project-01/Albosuites03.jpg)

## Tecnologías Usadas

- **Adobe Photoshop**: Para la edición y optimización de imágenes.
- **Adobe Illustrator**: Para la creación de elementos gráficos y composiciones visuales.
- **Cinema 4D**:: Para la generación de visuales en 3D que ayudaran a representar el proyecto de manera más atractiva.

## Diseño Web para Albosuites.ec

Lideré el diseño de la identidad visual del sitio web de Albo Suites, asegurando que la plataforma reflejara modernidad, accesibilidad y la esencia del proyecto inmobiliario. La estrategia se centró en transmitir la comodidad y ubicación estratégica del proyecto, optimizando la experiencia del usuario para facilitar la navegación y el acceso a la información clave sobre las suites.


![Albosuites web content 1](/images/projects/project-01/Albosuitesweb01.jpg)

# Retos Principales

- **Experiencia de usuario intuitiva**: Diseñar una interfaz clara y funcional que permitiera a los usuarios explorar fácilmente las opciones de vivienda.

- **Equilibrio entre estética y funcionalidad**: Crear una identidad visual atractiva sin sacrificar la accesibilidad y facilidad de navegación.

- **Optimización para múltiples dispositivos**: Garantizar una experiencia fluida tanto en computadoras como en dispositivos móviles.

![Albosuites web content 1](/images/projects/project-01/Albosuitesweb02.jpg)

## Tecnologías Usadas
- **Figma**: Herramienta clave en este proyecto para el diseño UI/UX, facilitando la creación de prototipos interactivos y la estructuración de una experiencia de usuario optimizada.

- **Adobe Photoshop**: Para la optimización de imágenes y diseño de elementos visuales en la web.

- **Adobe Illustrator**: Para la creación de iconografía y gráficos alineados con la identidad de la marca.
